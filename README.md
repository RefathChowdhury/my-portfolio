## Run Locally

### Clone the repo
```bash
git clone https://github.com/RefathChowdhury/my-portfolio.git
cd my-portfolio
```

### Install dependencies
```bash
npm install
```

### Start the dev server
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
